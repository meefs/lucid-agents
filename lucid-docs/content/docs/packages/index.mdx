## Architecture

```
┌─────────────────────────────────────────────────────────┐
│                    Developer Tools                       │
│                      @lucid-agents/cli                   │
└─────────────────────────────────────────────────────────┘
                            │
┌─────────────────────────────────────────────────────────┐
│                   Framework Adapters                     │
│         hono  │  tanstack  │  express  │  next          │
└─────────────────────────────────────────────────────────┘
                            │
┌─────────────────────────────────────────────────────────┐
│                      Extensions                          │
│   http │ payments │ identity │ a2a │ wallet │ ap2       │
└─────────────────────────────────────────────────────────┘
                            │
┌─────────────────────────────────────────────────────────┐
│                         Core                             │
│                    @lucid-agents/core                    │
└─────────────────────────────────────────────────────────┘
                            │
┌─────────────────────────────────────────────────────────┐
│                        Types                             │
│                   @lucid-agents/types                    │
└─────────────────────────────────────────────────────────┘
```

Dependencies flow downward. The core runtime is protocol-agnostic; HTTP is provided as an extension.
