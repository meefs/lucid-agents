Hosted agent creation may combine an MCP tool, a server wallet, generated
handlers, and platform deployment. That is a managed product workflow, not a
prerequisite for Lucid SDK development.

Before using it, confirm current availability, production MCP configuration,
wallet custody, supported networks, generated-code review, deployment limits,
data retention, and the process for exporting or deleting an agent.

To build locally with any coding agent, follow
[Build with a coding agent](/docs/start/coding-agent). That path uses repository
instructions, normal source control, and the same tests as hand-written code.
