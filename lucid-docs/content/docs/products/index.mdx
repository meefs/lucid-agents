Hosted products have their own availability, authentication, pricing, limits,
and support lifecycle. They are not required to build or deploy the open-source
SDK.

This is an availability index, not a claim that each product has a public
contract. A product becomes an integration destination only when its page can
name a supported production endpoint, authentication, pricing/limits, data
handling, service status, and support path. Until then, its page is explicitly
an ownership boundary.
